## 🎯 产品概述

**AI Director Service** 是一款面向短视频营销场景的智能化创作引擎，致力于帮助品牌方、内容运营团队及营销平台高效产出高质量短视频内容。

通过深度融合 **Qwen3.6-series 大语言模型** 与行业知识规则，本服务可自动完成「文案合规→内容优化→素材匹配→方案输出」的全链路决策，显著降低人工创作成本，提升内容生产效率与合规安全性。

### 🎁 核心价值

| 价值维度       | 具体收益                                     |
|------------|------------------------------------------|
| ⚡ 效率提升     | 单条短视频方案生成从小时级缩短至秒级，支持批量任务并行处理            |
| 🛡️ 合规保障   | 内置行业敏感词库与表达规范，自动拦截风险文案，降低审核返工率           |
| 🎨 创意赋能    | 基于大模型理解活动意图，智能匹配视频/模板/配音/BGM，激发创意组合      |
| 📊 策略可控    | 支持模板策略、重试机制、视频数量等业务参数配置，灵活适配不同场景         |
| 🔌 无缝集成    | 标准 RESTful API 设计，轻松对接内容中台、营销自动化系统或第三方平台 |

## ✨ 核心能力

### 🔍 文案合规检查
自动识别文案中的错别字、不规范表达、敏感词及行业红线内容，返回明确违规原因与修正建议，助力内容"零风险"上线。

### ✨ 智能文案优化
基于 Qwen3.6-series 的语义理解与生成能力，对原始文案进行风格润色、表达增强与场景适配，让内容更具吸引力与转化力。

### 🎞️ AI 智能选材
输入展会/活动基本信息（城市、名称、时间、地点、行业），自动从海量素材库中推荐匹配的视频片段、封面模板、背景音乐、配音角色及参考脚本，实现"输入需求，输出方案"。

### 🎁 一站式方案推荐
综合用户偏好、历史数据与业务策略，输出多套结构化短视频生成方案，包含开场视频、常规片段、视觉模板、音频配置与最终文案，直接对接视频合成引擎，实现端到端自动化生产。

## 🏗️ 技术亮点

- **大模型驱动**：基于 Qwen3.6-series 构建推理与生成核心，兼顾中文理解深度与生成质量
- **异步高性能**：FastAPI + async/await 架构，单实例支持高并发请求，响应延迟低
- **灵活可编排**：LangChain 实现 AI 决策流程模块化，便于后续扩展新能力或调整策略
- **容器化部署**：Docker 镜像交付，支持 Kubernetes 编排，适配私有化/混合云多种部署场景
- **可观测性友好**：结构化日志 + 健康检查接口，便于监控告警与问题排查

## 📦 典型应用场景

### 🎪 展会/活动营销
- **输入**：展会名称、时间、地点、行业、宣传要点
- **输出**：多套短视频方案，含开场视频+精彩片段+宣传文案+适配BGM
- **价值**：快速批量生成活动预热/现场回顾/招商推广视频

### 🛍️ 电商促销短视频
- **输入**：商品卖点、促销信息、目标人群
- **输出**：合规优化后的口播文案 + 匹配商品展示视频 + 节奏感BGM
- **价值**：提升商品视频生产效率，保障文案合规，降低投放风险

### 🏢 企业品牌内容生产
- **输入**：品牌调性、传播主题、素材库权限
- **输出**：符合品牌规范的系列短视频方案，支持多风格/多角色配音
- **价值**：统一品牌表达，规模化产出高质量内容
